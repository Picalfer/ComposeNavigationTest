[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Jetpack Compose](https://img.shields.io/badge/Jetpack-Compose-blue.svg?logo=android)](https://developer.android.com/jetpack/compose)

Тестовое приложение для изучения навигации в Jetpack Compose с использованием type-safe API.

## О проекте

Приложение демонстрирует современный подход к навигации в Compose с Kotlin Serialization. Включает bottom navigation и передачу данных между экранами.

**Основной функционал:**
- Bottom Navigation Bar с тремя вкладками
- Список элементов с переходом на детальный экран
- Type-safe навигация с передачей параметров
- Композable экраны

## Технологии

* Kotlin
* Jetpack Compose
* Compose Navigation
* Kotlin Serialization
* Material 3
